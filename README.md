# Jornadadev

Bem-vindo ao repositório **JornadaDev**! Este espaço foi criado para organizar e documentar todo o meu aprendizado no desenvolvimento de software, reunindo exercícios, códigos práticos e projetos de estudo.

## 📁 Estrutura do Repositório

O projeto está estruturado de forma simples e profissional para facilitar a navegação:

- **`Modulo01/`**: Contém os primeiros códigos-fontes desenvolvidos em Harbour (`.prg`) e C (`.c`).
- **`Modulo02/`**: Espaço reservado para as próximas etapas e novos aprendizados da jornada.
- **`assets/`**: Pasta destinada ao armazenamento de mídias e capturas de tela do progresso dos sistemas.

## 🛠️ Tecnologias Utilizadas

- **C** (Lógica de programação e conceitos estruturados)
- **Harbour** (Desenvolvimento de sistemas de dados e telas)
- **Git & GitHub** (Controle de versão e organização do código)
- **Visual Studio Code** (Ambiente de desenvolvimento)

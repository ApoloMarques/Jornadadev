# Exercício 01 - Função ou Procedimento

## 1. Str()

**Função**

Retorna uma string a partir de um número.

---

## 2. QOut()

**Procedimento**

Apenas exibe uma mensagem na tela, sem retornar um valor.

---

## 3. Date()

**Função**

Retorna a data atual do sistema.

---

## 4. Len()

**Função**

Retorna o tamanho de uma string ou de um array.

---

## 5. Upper()

**Função**

Retorna uma string convertida para letras maiúsculas.

---

## 6. ClearScreen()

**Procedimento**

Apenas limpa a tela, sem retornar nenhum valor.
